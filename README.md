# 淘宝用户行为分析

> 基于 20 万条用户行为日志的端到端数据分析项目，覆盖用户画像、转化漏斗、留存分析和用户分层。

## 业务背景

某电商平台希望**提升用户转化率和留存率**，但运营团队目前只关注整体 GMV，对用户行为路径缺乏系统认知。本分析旨在回答：

1. **用户整体行为模式是怎样的？** 什么时间活跃？行为类型如何分布？
2. **转化漏斗哪里卡住了？** 从浏览到购买的每一步流失率是多少？不同品类差异多大？
3. **用户会回来吗？** 次日/3 日/7 日留存率是多少？不同行为路径的用户留存差异？
4. **谁是好用户？** 能否按 RFM 思路将用户分层，指导差异化运营？
5. **如果我是运营负责人，我会做什么？** 基于数据给出 3 条可落地的建议。

## 数据说明

- 数据来源：基于阿里天池 UserBehavior 数据集字段结构，使用 Python 构造模拟数据
- 时间范围：2017-11-25 至 2017-12-08（14 天）
- 数据量：200,000 条行为记录，4,673 个用户，6,976 个商品，50 个品类

### 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| user_id | int | 用户 ID（脱敏） |
| item_id | int | 商品 ID（脱敏） |
| category_id | int | 品类 ID（脱敏，共 50 个） |
| behavior_type | str | 行为类型：pv(浏览) / fav(收藏) / cart(加购) / buy(购买) |
| time | datetime | 行为时间（精确到秒） |

## 关键发现

### 1. 用户画像
- **周末效应显著**：周末日均 UV 比工作日高 ~15%，行为集中在 9-23 点（典型电商作息）
- **用户行为呈长尾分布**：24% 的头部用户贡献了 80% 的行为量，中位数用户仅 14 次行为
- **整体转化率**：4,673 用户中 52.4% 有购买行为，但大部分用户仅浏览

### 2. 转化漏斗
- **最大流失点：浏览→收藏**——仅 ~5% 的用户会产生收藏行为
- **品类间转化率差异大**：高转化品类的购买率比低转化品类高数倍
- **「仅浏览」是绝对主流路径**——超过 70% 的用户-商品交互对止步于浏览

### 3. 留存分析
- **首日行为深度决定留存**：首日有购买行为的用户次日留存显著高于仅浏览用户
- **24 小时内是转化的黄金窗口**——大部分购买发生在首次浏览后的第一个 24 小时内
- **周末加入的用户留存略高**（更多闲暇探索时间）

### 4. 用户分层（RFM）
- **高价值用户占比最小，但贡献最多购买**
- **流失风险用户是需要唤醒的沉默资产**
- **「加购未买」是最大的可挽回空间**——用户已表明购买意图但未完成

## 业务建议

1. **缩短转化路径**：在浏览后立即展示「看了又买」推荐 + 限时优惠，抓住 24 小时转化黄金窗口
2. **差异化运营**：高价值用户推 VIP 权益，加购未买用户推定向优惠券，流失风险用户推大额召回券
3. **优化周末流量承接**：周末流量高峰时加大促销力度，配合时段热力图在晚 8-10 点投放活动
