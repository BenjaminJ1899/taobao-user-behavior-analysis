# 淘宝用户行为分析

> 基于 20 万条用户行为日志的端到端数据分析项目，覆盖用户画像、转化漏斗、留存分析和用户分层。

## 业务背景

某电商平台希望**提升用户转化率和留存率**，但运营团队目前只关注整体 GMV，对用户行为路径缺乏系统认知。本分析旨在回答：

1. **用户整体行为模式是怎样的？** 什么时间活跃？行为类型如何分布？
2. **转化漏斗哪里卡住了？** 从浏览到购买的每一步流失率是多少？不同品类差异多大？
3. **用户会回来吗？** 次日/3 日/7 日留存率是多少？不同行为路径的用户留存差异？
4. **谁是好用户？** 能否按 RFM 思路将用户分层，指导差异化运营？
5. **如果我是运营负责人，我会做什么？** 基于数据给出 3 条可落地的建议。

## 数据说明

- 数据来源：基于阿里天池 UserBehavior 数据集字段结构，使用 Python 构造模拟数据（含因果行为路径）
- 时间范围：2017-11-25 至 2017-12-08（14 天）
- 数据量：200,000 条行为记录，6,000+ 个用户，8,000+ 个商品，49 个品类

### 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| user_id | int | 用户 ID（脱敏） |
| item_id | int | 商品 ID（脱敏） |
| category_id | int | 品类 ID（脱敏） |
| behavior_type | str | 行为类型：pv(浏览) / fav(收藏) / cart(加购) / buy(购买) |
| time | datetime | 行为时间（精确到秒） |

## 关键发现

### 1. 用户画像
- **工作日活跃度更高**：工作日日均 PV 略高于周末，行为集中在 9-23 点（典型电商作息）
- **用户行为呈长尾分布**：少数高活跃用户贡献大部分行为，中位数用户行为次数远低于均值
- **购买渗透率**：6,000+ 用户中约 69% 有购买行为

### 2. 行为渗透与转化
- **浏览→加购→购买是最有效的转化路径**：用户路径分析显示加购后购买率远高于收藏后购买
- **约 84% 的用户-商品交互对止步于纯浏览**，加购未买是最大的可挽回空间
- **品类间转化率差异显著**：不同品类的购买转化率存在数倍差距，可针对性优化

### 3. 留存分析
- **次日留存 ~66%**：大部分用户第二天还会回来
- **7 日留存 ~23%**：约四分之一的用户一周后仍在活跃
- **首日行为深度预示留存**：首日有购买行为用户的留存显著高于仅浏览用户
- **24 小时内是转化黄金窗口**：多数购买发生在首次浏览后的短时间内

### 4. 用户分层（RFM）
- **高价值用户占比最小，贡献最多购买**
- **流失风险用户是需要唤醒的沉默资产**
- **「加购未买」是最大的可挽回空间**——用户已表明购买意图但未完成

## 数据局限性

- 数据由 Python 程序模拟生成，基于电商行业的公开统计分布参数
- 行为路径通过概率状态机生成（pv→fav→cart→buy），有因果结构但非真实用户数据
- 缺少交易金额、用户画像字段（性别/年龄/地域），限制了分析深度
- 所有结论应在真实数据上验证后再用于决策

## 业务建议

1. **缩短转化路径**：在浏览后立即展示「看了又买」推荐 + 限时优惠，推动加购行为
2. **差异化运营**：高价值用户推 VIP 权益，加购未买用户推定向优惠券，流失风险用户推大额召回券
3. **强化工作日转化**：工作日用户更活跃，集中资源在工作日晚间 8-10 点（高峰时段）投放活动
